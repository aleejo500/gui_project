# Projet TransParix

## Prérequis

* Java SE 7
* GNU Make
* Driver JDBC

## Compiler le projet
Pour compiler, utilisez la commande suivante :
    make

## Lancer l'application
Pour executer le programme, utilisez :
    make exec

## Supprimer les fichiers générés
Pour supprimer les fichier *.class générés par la compilation :
    make clean
